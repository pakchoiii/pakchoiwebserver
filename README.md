#### 1、编译准备工作

post.html 以及 post.cgi需要一定的权限才可以执行。

1、cd  httpdocs

2、sudo chmod 600 test.html

sudo chmod 600 post.html

sudo chmod +X post.cgi

4、make

5、 ./myhttp
